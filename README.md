## 🎯 Featured Project

<div align="center">

### 🎬 PentaPlex — Multi-Branch Cinema Management System

</div>

> **SWP391 Capstone** · Hệ thống quản lý rạp chiếu phim đa chi nhánh, 5 vai trò (Guest / Customer / Branch Manager / Branch Staff / Admin).

| | |
|---|---|
| **Đặt vé online & tại quầy** | Chọn ghế real-time (WebSocket), giữ ghế PENDING, hết hạn tự huỷ |
| **Thanh toán** | VNPay (HMAC-SHA512 callback) + tiền mặt tại quầy |
| **Vé điện tử** | QR code (ZXing) + PDF (iText 7) + email xác nhận (JavaMail) |
| **Quản lý** | Phim, lịch chiếu (tạo hàng loạt), phòng/ghế, giá vé, F&B, khuyến mãi, báo cáo doanh thu |
| **Kiến trúc** | Java 17 · Servlet/JSP · JDBC + DAO · SQL Server · Filter chain (Auth/Role/Branch) |
